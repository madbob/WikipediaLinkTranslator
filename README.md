# Wikipedia Link Translator

### Installation

Requirements:

 * PHP >= 5.6
 * Composer >= 1.0
 * Redis >= 3.0

```
git clone https://github.com/madbob/WikipediaLinkTranslator
cd WikipediaLinkTranslator
composer install
```

### Public Instance

You can find a public instance at http://wlt.madbob.org/
